Dimensions on the Morgan-Scott Split

The table on this page illustrates in more detail the dependence of the dimension of a spline space on the geometry of the underlying triangulation. We consider two configurations of the Morgan-Scott split (Morgan and Scott, 1975) as illustrated in the nearby figure.

The split shown in the figure is symmetric, the second generic. Specifically, in the symmetric split,

V4 = (V1 + 2 V2 + 2 V3)/5,  

V5 = (2V1 + V2 + 2 V3)/5,  

V6 = (2V1 + 2V2 + V3)/5.

It is essential that these equalities hold exactly and the MDS code does make sure of that.

By comparison, in the generic version,

V4 = (10V1 + 20V2 +20V3)/50,  

V5 = (21V1 + 10V2 +22V3)/53,  

V6 = (20V1 + 17V2 + 10V3)/47.
